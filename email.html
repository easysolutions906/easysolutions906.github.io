Frequently Asked Questions

What does this email validator check?

Five checks: (1) syntax validation against RFC 5322, (2) MX record lookup to verify the domain accepts email, (3) disposable email detection against 3,000+ throwaway domains, (4) free provider detection (Gmail, Yahoo, Outlook, etc.), and (5) an overall deliverability score from 0-100.

What is a disposable email?

Disposable emails are temporary addresses from services like Guerrilla Mail, Temp Mail, or Mailinator. They are often used to bypass signups. Detecting them helps keep your user database clean.

Does this tool send an email to verify the address?

No. Validation is done through DNS lookups and pattern matching. No emails are sent, so there is zero risk of triggering spam filters or bouncing.

Is there a batch validation API?

Yes. The API supports batch validation of up to 50 emails per request via POST /validate/batch. Use any paid plan for batch access.
